Detailed Description

a view container which draws into a platform layer on top of a parent layer or the platform view

A CLayeredViewContainer creates a platform layer on top of a parent layer or the platform view of CFrame if available on that platform and draws into it, otherwise it acts exactly like a CViewContainer
